Welcome!

This ie_pandas package includes pandas-like functionalities for the following data structures:
	-DataFrames

Important info:
-The package has been developed with Python 3.7.2
-Once the source code is downloaded you can install the package using pip install. Ex: 'pip install ie_pandas'
-To run the unit tests (located in the tests folder) use pytest. Ex: 'pytest test_ie_pandas.py'
-If you have any contributions feel free to create pull requests and we will consider your suggestions!
